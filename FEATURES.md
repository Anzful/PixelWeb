- рЃцрЃћрЃарЃЌрЃљ рЃърЃљрЃџрЃўрЃбрЃарЃљ:
  - Primary: рЃџрЃБрЃарЃ»рЃў (#0ea5e9)
  - Accent: рЃЮрЃЦрЃарЃЮрЃАрЃцрЃћрЃарЃў (#d97706)
  - рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃюрЃћрЃўрЃбрЃарЃљрЃџрЃБрЃарЃў рЃбрЃЮрЃюрЃћрЃЉрЃў

## ­ЪЊ▒ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃћрЃЦрЃфрЃўрЃћрЃЉрЃў

- рЃЏрЃЮрЃЎрЃџрЃћ рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃў value proposition
- рЃЮрЃарЃў CTA рЃдрЃўрЃџрЃљрЃЎрЃў (рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ)
- рЃљрЃюрЃўрЃЏрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљ (4 рЃЏрЃћрЃбрЃарЃўрЃЎрЃљ)

**Features Section:**

- Card-based рЃЊрЃўрЃќрЃљрЃўрЃюрЃў hover рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃўрЃЌ
- Staggered рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў scroll-рЃќрЃћ

**Pricing Preview:**
- 3 рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃърЃљрЃЎрЃћрЃбрЃўрЃА preview

- рЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃГрЃЋрЃўрЃарЃЋрЃљрЃџрЃћ рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ
- рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃў

**CTA Section:**

- рЃЊрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃЮрЃгрЃЮрЃЊрЃћрЃЉрЃљ

### 2. рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў (/services)
**рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃцрЃљрЃАрЃћрЃЉрЃў:**

- рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃАрЃљрЃўрЃбрЃў: 800-1,200 РѓЙ

**рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃљрЃЎрЃћрЃбрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА:**

- рЃЉрЃљрЃќрЃарЃўрЃА рЃцрЃљрЃАрЃЌрЃљрЃю рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃљрЃА

- рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ
- рЃцрЃЮрЃбрЃЮрЃњрЃарЃљрЃцрЃўрЃљ
- рЃЏрЃћрЃюрЃбрЃћрЃюрЃљрЃюрЃАрЃў

**рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃърЃарЃЮрЃфрЃћрЃАрЃў:**

### 3. рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ рЃњрЃЋрЃћрЃарЃЊрЃў (/portfolio)

  - рЃърЃўрЃарЃљрЃЊрЃў/рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮ
  - рЃ»рЃљрЃюрЃЊрЃљрЃфрЃЋрЃљ
  - рЃБрЃФрЃарЃљрЃЋрЃў рЃЦрЃЮрЃюрЃћрЃЉрЃљ
